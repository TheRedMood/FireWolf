Installation &c.

1) Run ./firewolf.py - this creates a default config file
2) Edit default.py
3) Run ./firewolf.py - this now runs firewolf with your settings

NOTE:
If you want to use the databases and so on you need to go into data and look around.

data/ro_itemdb.py - generates a sql database of the items in the game ragnarok online
Enjoy!
